Get Zoi installed and working in three simple steps. This guide will take you from installation to your first package install.

<Steps>
  <Step title="Install Zoi">
    Choose the installation method that works best for your system:

    <Tabs>
      <Tab title="Linux / macOS">
        Use the installer script for a quick setup:

        ```bash
        curl -fsSL https://zillowe.pages.dev/scripts/zoi/install.sh | bash
        ```

        <Accordion title="Alternative: Package managers">
          **Arch Linux (AUR)**
          ```bash
          yay -S zoi-bin
          ```

          **Homebrew (macOS/Linux)**
          ```bash
          brew install Zillowe/tap/zoi
          ```

          **Cargo**
          ```bash
          cargo install zoi-rs
          ```
        </Accordion>
      </Tab>

      <Tab title="Windows">
        Use the PowerShell installer:

        ```powershell
        powershell -c "irm zillowe.pages.dev/scripts/zoi/install.ps1|iex"
        ```

        <Accordion title="Alternative: Scoop">
          ```powershell
          scoop bucket add zillowe https://github.com/Zillowe/scoop.git
          scoop install zoi
          ```
        </Accordion>
      </Tab>
    </Tabs>

    <Tip>
      After installation, restart your terminal or run `source ~/.bashrc` (or equivalent for your shell) to update your PATH.
    </Tip>

    Verify the installation:

    ```bash
    zoi version
    ```

    <Accordion title="Example output">
      ```
      zoi 1.7.0 (Production/Release)
      Commit: a1b2c3d
      ```
    </Accordion>
  </Step>

  <Step title="Sync Package Repositories">
    Before you can install packages, sync the official package database from the Zoidberg registry:

    ```bash
    zoi sync
    ```

    This command:
    - Clones the Zoidberg package registry to `~/.zoi/pkgs/db/`
    - Sets up shell completions automatically
    - Checks for available package managers on your system

    <Accordion title="Example output">
      ```
      Syncing registries...
      ✓ zoidberg: Cloned successfully (1.2s)
      ✓ Detected package managers: apt, snap, cargo, npm, pip
      ✓ Shell completions installed for bash
      
      Synced 1 registry with 2,847 packages
      ```
    </Accordion>

    <Info>
      Zoi supports multiple registries. You can add custom registries with `zoi sync add <url>`. See the [repositories guide](/repositories) for details.
    </Info>
  </Step>

  <Step title="Install Your First Package">
    Now you're ready to install packages. Let's install the `hello` example package:

    ```bash
    zoi install @zillowe/hello
    ```

    Zoi will:
    1. Resolve dependencies
    2. Download the package for your platform
    3. Verify the checksum
    4. Install it to the Zoi store
    5. Create symlinks in your PATH

    <Accordion title="Example output">
      ```
      Resolving dependencies...
      ✓ Resolved hello@1.0.0 in 127ms
      
      Downloading packages...
      ✓ hello@1.0.0 (linux-amd64) [1.2 MB] (534ms)
      
      Verifying packages...
      ✓ Verified checksum (SHA512)
      
      Installing packages...
      ✓ Installed hello@1.0.0 to ~/.zoi/pkgs/store/hello-1.0.0
      ✓ Linked bin/hello → ~/.zoi/pkgs/bin/hello
      
      Successfully installed 1 package
      ```
    </Accordion>

    Test the installed package:

    ```bash
    hello
    ```

    You should see:
    ```
    Hello from Zoi!
    ```

    <Note>
      If the `hello` command isn't found, make sure `~/.zoi/pkgs/bin` is in your PATH. Run `zoi shell <your-shell>` to configure your shell automatically.
    </Note>
  </Step>
</Steps>

## Common Tasks

Now that Zoi is set up, here are some common commands you'll use:

### Search for Packages

```bash
zoi search editor
```

Use the interactive mode for a better experience:

```bash
zoi search editor -i
```

### List Installed Packages

```bash
zoi list
```

### Show Package Details

```bash
zoi show @zillowe/hello
```

### Update Packages

Update a specific package:

```bash
zoi update @zillowe/hello
```

Update all packages:

```bash
zoi update --all
```

### Uninstall a Package

```bash
zoi uninstall @zillowe/hello
```
