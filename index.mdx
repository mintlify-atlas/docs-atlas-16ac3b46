  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Zoi up and running in three simple steps
  </p>

  <Steps>
    <Step title="Install Zoi">
      Choose your preferred installation method:

      <CodeGroup>
      ```bash Linux / macOS
      curl -fsSL https://zillowe.pages.dev/scripts/zoi/install.sh | bash
      ```

      ```powershell Windows
      powershell -c "irm zillowe.pages.dev/scripts/zoi/install.ps1|iex"
      ```

      ```bash Homebrew
      brew install Zillowe/tap/zoi
      ```

      ```bash Cargo
      cargo install zoi-rs
      ```
      </CodeGroup>

      <Tip>
        On Arch Linux, install from the AUR: `yay -S zoi-bin`
      </Tip>
    </Step>

    <Step title="Sync Package Repositories">
      Before installing packages, sync the official package database:

      ```bash
      zoi sync
      ```

      This command clones the Zoidberg package registry to your local machine and sets up shell completions.
    </Step>

    <Step title="Install Your First Package">
      Now you're ready to install packages:

      ```bash
      zoi install @zillowe/hello
      ```

      Zoi resolves dependencies, downloads the package, verifies checksums, and installs it to your system. The `hello` command is now available in your PATH.

      <Accordion title="Example output">
        ```
        ✓ Resolved dependencies in 234ms
        ✓ Downloaded hello@1.0.0 (linux-amd64)
        ✓ Verified checksum (SHA512)
        ✓ Installed to ~/.zoi/pkgs/store/hello-1.0.0
        ✓ Linked bin/hello → ~/.zoi/pkgs/bin/hello
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
